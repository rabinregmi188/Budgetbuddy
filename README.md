# BudgetBuddy

BudgetBuddy is a responsive personal finance dashboard for tracking income and expenses, setting category budgets, and analyzing monthly spending patterns.

## Features

- Add income and expense transactions with categories and dates
- Set monthly budgets for each category
- Filter all data by month
- View summary cards for income, expenses, balance, and budget usage
- See category spending bars and budget progress tracking
- Export transaction history to CSV
- Persist data locally in the browser with `localStorage`

## Stack

- HTML
- CSS
- JavaScript

## Run locally

Open `index.html` in your browser.
